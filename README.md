# Movie Finder (TypeScript Practice Project)

This project was built as a TypeScript practice application.
It fetches movie data from the OMDb API, renders results as UI cards,
and stores recent searches in localStorage.

---

## Features

- Search movies by title
- Optional filter by year
- Display results in responsive card layout (poster, title, year)
- Save recent searches to localStorage
- Prevent duplicate saved searches

---

## Tech Stack

- TypeScript
- Axios
- Webpack
- HTML
- CSS

---

## Running the Project

Install dependencies:

npm install

Start development mode:

npm run dev

---

- Working with external APIs
- Creating and using custom TypeScript interfaces

- Using localStorage with duplicate prevention logic
